# 八、推荐工具链

## 导入来源：二进制协议解析指南.md

## 八、推荐工具链

| 工具 | 用途 | 平台 |
|------|------|------|
| **Wireshark** | 网络/串口 协议抓包分析 | 全平台 |
| **pyserial** | Python 串口通信 | 全平台 |
| **serialplot** | 串口数据实时波形显示 | Windows/Linux |
| **PuTTY / minicom** | 串口终端调试 | 全平台 |
| **Modbus Poll** | Modbus 主站调试工具 | Windows |
| **cangaroo / SavvyCAN** | CAN 总线分析 | 全平台 |
| **MQTT Explorer** | MQTT 可视化调试 | 全平台 |
| **HHD Serial Monitor** | 高级串口监控 (支持协议解析) | Windows |
| **Logic Analyzer (Saleae)** | 逻辑分析仪 (硬件，几十块) | 全平台 |
| **protobuf-inspector** | Protobuf 裸数据查看 | pip install |
