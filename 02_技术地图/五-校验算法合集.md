## 五、校验算法合集

### 5.1 常用校验

```python
# ===== XOR Checksum (最简单) =====
def xor_checksum(data: bytes) -> int:
    result = 0
    for b in data:
        result ^= b
    return result

# ===== Sum Checksum (8-bit) =====
def sum8_checksum(data: bytes) -> int:
    return sum(data) & 0xFF

# ===== CRC8 (NMEA 0183 使用) =====
def crc8_nmea(data: bytes) -> int:
    """CRC-8: 对 '$' 和 '*' 之间的所有字符异或"""
    result = 0
    for b in data:
        result ^= b
    return result

# ===== CRC16-Modbus (最常用) =====
def crc16_modbus(data: bytes) -> int:
    crc = 0xFFFF
    for byte in data:
        crc ^= byte
        for _ in range(8):
            if crc & 0x0001:
                crc = (crc >> 1) ^ 0xA001
            else:
                crc >>= 1
    return crc  # 小端发送: [crc_low][crc_high]

# ===== CRC16-CCITT (MAVLink 使用) =====
# MAVLink 使用 CRC16-CCITT with seed 0xFFFF
MAVLINK_CRC_EXTRA_TABLE = {
    # message_id → crc_extra (参考 MAVLink 协议文档)
    0:  50,   # HEARTBEAT
    1:  124,  # SYS_STATUS
    30: 39,   # ATTITUDE
    33: 67,   # GLOBAL_POSITION_INT
    # ... 完整表参考 MAVLink 官方
}

def crc16_ccitt(data: bytes, seed: int = 0xFFFF) -> int:
    crc = seed
    for byte in data:
        crc ^= (byte << 8)
        for _ in range(8):
            if crc & 0x8000:
                crc = ((crc << 1) ^ 0x1021) & 0xFFFF
            else:
                crc = (crc << 1) & 0xFFFF
    return crc

# ===== CRC32 (Ethernet, PNG 使用) =====
import binascii
crc32_result = binascii.crc32(data) & 0xFFFFFFFF
```

### 5.2 校验算法选择指南

```
场景                      推荐校验
─────────────────────────────────────
GPS NMEA 0183             XOR 后取 Hex (如 *3F)
Modbus RTU/TCP            CRC16-Modbus
MAVLink                   CRC16-CCITT
MQTT                      TCP 层已有校验，应用层无需
CAN 总线                  CAN 控制器硬件 CRC15
自定义简单协议              XOR Checksum (够了)
自定义可靠性要求高          CRC16 (工业级)
```
